# MaffContaby Brand Book

## Visão Geral

**MaffContaby** é uma plataforma moderna de controle contábil pessoal e familiar. Desenvolvida para simplificar o rastreamento de despesas, receitas e investimentos, a marca reflete **confiabilidade, clareza e sofisticação**.

A identidade visual combina elementos de design contemporâneo com uma abordagem acessível, criando uma experiência que inspira confiança e facilita a gestão financeira.

---

## 1. Logo & Marca

### Conceito
O logo integra a letra "M" (de MaffContaby) com um gráfico de barras ascendente, simbolizando crescimento financeiro e análise de dados. A forma circular envolvendo o gráfico representa segurança e integridade.

### Variações
- **Logo Principal**: Marca completa com nome (horizontal)
- **Ícone**: Apenas o símbolo "MC" para favicon e atalhos
- **Monocromático**: Versão em preto/branco para aplicações restritas

### Espaçamento Mínimo
Manter uma zona de proteção de 20px ao redor do logo em todos os usos.

## 8. Animações

### Transições Padrão
- **Rápida**: 150ms ease-out (hover, focus)
- **Normal**: 200ms ease-in-out (estado, mudança)
- **Lenta**: 300ms ease-in (entrada, saída)

### Easing Functions
- **ease-out**: Entrada suave
- **ease-in-out**: Transição natural
- **ease-in**: Saída suave

### Animações Principais
- **Fade In**: Opacidade 0 → 1 (300ms)
- **Slide Up**: Transform translateY(20px) → 0 (300ms)
- **Scale**: Transform scale(0.95) → 1 (200ms)
- **Pulse**: Opacidade 1 → 0.5 → 1 (2s infinite)

## 12. Guia de Uso

### Quando Usar Cada Cor
- **Azul Marinho**: Headers, CTAs primárias, elementos de destaque
- **Verde Esmeralda**: Sucesso, crescimento, valores positivos
- **Vermelho Coral**: Erros, valores negativos, alertas críticos
- **Cinza**: Texto, ícones, elementos secundários

### Combinações Recomendadas
- Azul + Verde: Dashboards de análise
- Azul + Vermelho: Comparações (entrada vs. saída)
- Verde + Âmbar: Indicadores de saúde financeira

### Evitar
- Usar apenas cores para diferenciar (adicionar ícones/texto)
- Combinações com baixo contraste
- Mais de 3 cores primárias em uma tela
- Gradientes muito vibrantes (manter sutileza)
